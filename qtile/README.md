# Qtile

This folder contains the configuration for the Qtile window manager.

- `config/` stores the main config (`config.py`) plus settings and theme data.
- `wallpapers/` holds wallpapers used by the Qtile setup.
- `xprofile` records the stable X11 fallback startup configuration used by the existing host.

See `docs/qtile.md` for the temporary fallback policy, dynamic dock handling, and verification steps.
